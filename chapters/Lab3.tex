 \section{Assignment}
 After learning about various scheduling policies and blocking tasks, the assignment for Lab 3 focuses on writing a short program that calculates worst case response time using \textit{Deadline Monotonic Scheduling}.

The algorithm tasks the task list and TaskID to calculate the time for as input and uses the formula \ref{math:response_time} to calculate the worst case response time. The algorithm iteratively calculates the response time until it converges or exceeds the deadline.

\label{math:response_time}
\[
  R^{n+1}_i = C_i + \sum_{K \in hp(i)} \left\lceil \frac{R^n_i}{T_K} \right\rceil C_K
\]

\begin{center}
\begin{tabular}{l}
\textbf{Algorithm: Worst-Case Response Time (WCET)} \\[4pt]
\hline \\[-6pt]
\textbf{Input:} Task set $\mathcal{T}$, task $\tau_k$ \\ 
\textbf{Output:} Worst-case response time $R_k$ \\[6pt]

1.\quad Select task $\tau_k \in \mathcal{T}$ \\

2.\quad $\mathcal{HP} \leftarrow
\{\tau_i \in \mathcal{T} \mid \text{priority}(\tau_i) >
\text{priority}(\tau_k)\}$ \\[4pt]

3.\quad $R_{\text{prev}} \leftarrow -1$ \\

4.\quad $R \leftarrow 0$ \\[4pt]

5.\quad \textbf{while} $R \neq R_{\text{prev}}$ \textbf{do} \\
\quad 5.1\quad $R_{\text{prev}} \leftarrow R$ \\
\quad 5.2\quad $R \leftarrow C_k$ \\
\quad 5.3\quad \textbf{for each} $\tau_i \in \mathcal{HP}$ \textbf{do} \\
\quad\quad $R \leftarrow R +
\left\lceil \dfrac{R_{\text{prev}}}{T_i} \right\rceil C_i$ \\
\quad 5.4\quad \textbf{end for} \\

6.\quad \textbf{end while} \\[4pt]

7.\quad \textbf{return} $R$ \\
\end{tabular}
\end{center}
